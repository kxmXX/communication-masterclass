# Founder Copy — Quick Start

## Cas le plus fréquent : "J'ai besoin d'un post LinkedIn qui vend mon projet"

### Solution en 3 étapes :

1. **Demande ces 3 infos** (tout le reste est déduit) :
   - "Ton projet en UNE phrase ?"
   - "Quel est le résultat que t'as obtenu ?" (chiffre ou signal fort)
   - "Qu'est-ce qui rend ce projet possible MAINTENANT ?" (Why Now)

2. **Structure standard** :
   ```
   [Micro histoire personnelle de 2-3 lignes]
   [Le problème que ça révèle]
   [Les chiffres / la preuve]
   [La leçon universelle en 1-3 points max]
   [Question ouverte]
   ```

3. **Rappels** :
   - "I" > "We". Sauf Japon.
   - 200-400 mots. Pas un blog post.
   - Un chiffre tous les 2-3 paragraphes.
   - Termine par une question.
   - Relis à voix haute.

## Mode "Rewrite"

L'utilisateur a déjà un texte. Il veut que tu l'améliores.

1. Analyse ce qui marche DANS SON TEXTE
2. Identifie CE QUI MANQUE : hook ? chiffres ? CTA ? Why Now ?
3. Produis V1 améliorée EN GARDANT SON STYLE
4. Auto-critique (le nouveau texte doit sonner comme lui, pas comme toi)

## Cas rares (mais importants)

- **"Full campaign"** : besoin d'un post LinkedIn + une landing page + un teaser Twitter/X. Produis les 3 en une fois, cohérents.
- **"PR announcement"** : levée de fonds / partenariat / lancement. Format : titre accrocheur + 3 paragraphes + quote fondateur + 1 chiffre clé.
- **"Trouver mon ton"** : l'utilisateur sait pas quel ton adopter. Propose 3 options (ambitieux, humble, provocateur) avec 3 phrases d'exemple.
