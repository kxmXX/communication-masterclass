1. Ouvre `guide-pratique-fondateur.md`

## 🎯 Ce que ça sait faire

- **Posts LinkedIn** qui captent l'attention (calibrés par pays)
- **Pages Ulule/Kickstarter** avec la structure qui convertit ($220M+ analysés)
- **Pitch decks** slide 1-2 qui retiennent les investisseurs
- **Fundraising emails** qui ouvrent des portes
- **Landing pages** qui vendent sans bullshit
- **Communication de crise** qui désarme

## 🌍 Calibrage culturel

Le skill adapte automatiquement le ton selon ton pays :
- **USA** : Direct, ambitieux, vulnérabilité stratégique
- **France** : Subtil, ironique, contre-narrative
- **UK** : Self-deprecating, wit, understatement
- **Japon** : "We" pas "I", humble, collectif
- **Chine** : Visionnaire, chiffres massifs
- **Espagne** : Passionné, émotionnel
